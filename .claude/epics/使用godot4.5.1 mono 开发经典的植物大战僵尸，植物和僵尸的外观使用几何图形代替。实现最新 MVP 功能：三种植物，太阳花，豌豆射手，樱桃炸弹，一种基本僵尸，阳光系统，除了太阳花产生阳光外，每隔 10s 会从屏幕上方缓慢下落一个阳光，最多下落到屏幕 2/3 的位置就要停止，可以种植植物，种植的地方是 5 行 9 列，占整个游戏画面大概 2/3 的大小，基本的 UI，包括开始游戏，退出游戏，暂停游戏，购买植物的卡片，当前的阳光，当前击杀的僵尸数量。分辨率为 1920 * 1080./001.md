# Task: 项目基础搭建

## Description
初始化 Godot 4.5.1 Mono 项目，建立基础项目结构和开发环境配置，为后续开发奠定坚实基础。

## Acceptance Criteria
- [ ] 创建新的 Godot 4.5.1 Mono 项目，配置项目名称和基本设置
- [ ] 设置 C# 项目配置，包括目标框架 net8.0 和程序集名称
- [ ] 创建基础的场景结构：主菜单场景、游戏场景、UI场景
- [ ] 配置项目分辨率固定为 1920x1080
- [ ] 建立基础的脚本目录结构和命名空间规范
- [ ] 验证 GDAI MCP 插件正常工作并能够通过 AI 操作项目

## Technical Details

### 项目配置
- 使用 Godot 编辑器创建新项目
- 项目设置中启用 C# 支持
- 配置项目.godot文件，设置显示窗口大小
- 确保 C# 项目文件正确引用 .NET 8.0

### 目录结构
```
植物大战僵尸/
├── Scenes/           # 场景文件
│   ├── MainMenu.tscn
│   ├── GameScene.tscn
│   └── UI/
├── Scripts/          # C# 脚本
│   ├── Core/         # 核心系统
│   ├── Game/         # 游戏逻辑
│   ├── Plants/       # 植物
│   ├── Zombies/      # 僵尸
│   └── UI/           # 用户界面
├── Resources/        # 资源文件
└── project.godot     # 项目配置
```

### 基础场景创建
- **MainMenu.tscn**: 包含开始游戏和退出按钮的基础布局
- **GameScene.tscn**: 5x9网格区域和基础UI布局的占位符
